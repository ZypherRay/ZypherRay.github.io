“社会“与”公共“新冠肺炎疫情的社会学思考

怎么看：社会学的关切

从现象出发

  • 公共——社会

    • 新冠肺炎疫情作为一个公共卫生问题，无论防治还是修复，其着力点均在”公共“

    • ”公共“作为一种社会交往模式，其发生、运作和效应，均与一个社会的基本格局密切相关

  • 问题主要 不在医学治疗，而在社会治理

差序格局

  • 费孝通——《乡土中国》
    • ”一说是公家的，差不多就是说大家可以占一点便宜的意思，有权利而没有义务了。“
    • 这种现象，说明我国居民缺乏公共精神

现实需要

理解社会性——促进公共性

  • 正确认识”社会“的地位，更好地发挥”社会“的作用，预防社会衰退
  • 社会衰退并未引起广泛注意，一些旨在促进发展的促使，甚至会造成社会衰退
  • 社会衰退不比经济衰退危害更小，甚至更大，但是因为是长期病、慢性病、所以更难察觉，也很容易被忽视

社会衰退

  • 脱钩——链接
  • 梗阻——畅通
  • 排斥——包容
  • 压制——团结

四个故事

  1. 网课：教育革命？

    与实体课效果的对比，效果很难了解

    互动受限，传达信息有限

    【脱节】

  2. 买车：报复性消费？

    限购，车辆过户手续

    【梗阻】

  3. 口罩：”要死一起死“？

    居民预约领口罩，排斥外地人

    【排斥】

  4. 消费券：发发发?

    社会团结和公正问题，限制消费范围

    【压制】

每天一点社会学

  • 学校不只是教学场所，更是一种社会氛围
  • 脱离群体久了，个人也就蔫了
  • 每天都说要重新开始，却总觉得有些无力
  • 凝心聚气，不是靠闭门修炼，而是社会连接和团结
  • 社会就像空气一样，无处不在，却又无迹可寻

认识社会，认识社会学

  • 从正常中认识社会，从反常中认识社会

  • 从有常中认识社会，从无常中认识社会

怎么算：社会学的思维

理论问题

  1. 社会是什么？”社会“在哪里？
  2. ”社会“应当怎样治理，才更有弹性、韧劲和活力?
  3. “社会”应当怎样治理，才能有效地提高公共性，从而提高整个社会的凝聚力和生产力？

社会是什么

社会的形成

  • 社会作为人类的一种存在放视，是现代化的产物
  • 在传统社会中，社会的体量非常小，以致没有社会这一概念，更不可能形成社会学这样一门专门的学科
  • $地理上可流动*接触上有自由=社会$
  • 市场化、理性化、自由化

群体主义

  • 社会是人与人相互关联的产物
    1. 相互关联是人类存在的基本形式，人的生存和发展无往而不在人与人的关联之中
    2. 人生于，也长于特定的关联；不是先有个体，然后才形成某种关联，而是先有关联，然后形成个体
    3. 与他人的关联同时塑造着一个人的主观世界和客观世界
    4. 应当从人与人关联或”相交”处中去寻找和思考”社会“

社会在哪里

  • 衔接处把人类生活的各种要素、各个部分衔接起来而形成的整个系统【大社会】
  • 角落里归完类、点完名之后剩下的那些不想归类和点名，甚至想不起来的”其他“的集合【小社会】

“社会”的形态

“社会”的力量

  • 人与人是关联的，千丝万缕、牵牵绊绊，受不了、甩不掉

  • 社会关联模式是人类赖以开展政治、经济、文化等一切活动的”元代码“

  • ”元代码“的功能怎样才更强大？怎样关联才更和谐？

从社会性到公共性

社会性作为治理的难题

  • 数量庞大
  • 相互陌生
  • 随时结合
  • 无有定向

公共性作为治理方向

  • 对陌生人的尊重、负责、信任、体谅和照顾
  • 公共性的核心是怎样理解和处理陌生人问题

怎样对待陌生人，就是怎样对待你自己。在现代社会中，每个人随时可能成为陌生人

乡土社会

【差序格局】

以情感为基础建立和维持社会交往

  • 以”己“为中心，根据交情的远近形成一层套一层的圈子
  • 通过”拟亲“扩展社会交往”一表三千里“
  • 圈子的大小根据中心的势力而盈缩

公共性稀缺

  • 崇私德而薄公德：“系维着私人的道德”
  • 讲感情而非规则，作私民而非公民
  • 欺生：对陌生人怀疑、排斥甚至霸凌

单位社会

  • 每个人都以“单位”为中介与他人打交道
  • 每个人都是有单位、有身份的人，独立面对陌生的人时候几乎没有

单位行很强，公共性不足

新形势

各种社会要素都在时刻不停的流动和重新配置：

  • 人口
  • 资源
  • 社会地位
  • 价值期待

新形态

国家——【中间社会】——家庭、家族、村落、社区、居委、单位……

怎么干：社会学的办法

怎么组织

散众——合众——和众

怎么办

基于社会性，培养公共性

  • 保持四种形态的合理分界，不能夸大聚众和散众的力量，也不能压制

加强和创新社会治理

  • 逐渐实现“两个转变”

    1. 小治理——大治理

    2. 为社会——靠社会

      依靠群众，发动群众

  • 当前仍应以人民最关心最直接最现实的民生问题为重点，但要注意与远期的衔接，要为“两个转变”创造条件，起码要相向而行，而不能背道而驰。

”三中全会“【目的】

有机社会

  • 兼顾个人与社会、兼有秩序和活力
    1. 包容性：能够吸纳更多社会力量参与现代化建设
    2. 灵活性：能够敏捷而有效地应对来自外部坏境的各种挑战
    3. 柔韧性：能够应对和化解多个方面、多种性质的矛盾冲击
  • 社会学家的历史思考
    1. 马克思：人的异化
    2. 滕尼斯：群落——社会
    3. 韦伯：传统理性——工具理性
    4. 涂尔干：机械团结——有机团结
    5. 波兰尼：市场与社会的双重运动

回望四十年

搞活市场【富】——厚植社会【仁】

提问

什么是公共精神？

【核心】怎么对待陌生人

怎么量化社会脱节

  • 交往频次
  • 交往内容
  • 社会排斥
  • 社会压制

社会治理需要社会组织的参与吗

  • 需要

  • 热心网友——冷漠路人

互联网究竟是聚合了社会还是离散了社会

  • 互联网形态复杂，内容丰富，很难一概而论

现在是否还符合差序格局的特点

  • 现实还是较为重人情，规则意识有提升
  • 差序格局正在衰退
