高数总结

发表于 2020-11-17 | 分类于 -考研数学
字数统计: 2.1k | 阅读时长 ≈ 8

考研数学总结

高数

  1. 选择题如果计算量偏大，则需要考虑是不是方法错误，一般选择题计算量适中

  2. 计算时，不要先将参数的表达式代入，而是直接代参数计算，最后将参数带入【限于参数的表达式也是多项式】

  3. 当计算时，参数要代入具体值，则将能够代入得具体值先代入计算，会使计算变得简单

极限

计算

  1. 洛必达失效情况——极限不存在
  2. \(1^∞\)——\((u-1)*v\)
  3. 二元积分洛必达求极限，对x求导两次消掉积分号，两次分别对应两个不同的字母即可

\[ \lim_{n\rightarrow0}\frac{-\int_{0}^{x}du\int_{0}^{u^2}f(t,u)dt}{x^3} \]

\[ \lim_{n\rightarrow0}\frac{-\int_{0}^{x^2}f(t,x)dt}{3x^2} \]

  1. 两个字母注意使用换元，可以将定积分转换为变限积分
  2. 等价无穷小公式
  3. 夹逼准则的使用
  4. 分式的拆分，分子比分母始终少一阶

等价无穷小

  1. 微分方程与等价无穷小的结合
  • 设\(y=y(x)\)是\(y''+2y'+y=e^x\)满足\(y(0)=y'(0)=0\)的解，则当\(x→0\)时，与\(y(x)\)等价无穷小的是 \(1-cosx\)
  1. 复合函数的等价无穷小
  • 若\(f(x)——x^2\)则\(f(x^2)——x^4\)
  1. 等价无穷小可以用来判断极限的敛散性
  2. 注意使用泰勒公式
  3. \(ln(x+\sqrt{1+x^2)} ——x\)
  4. \(x-ln(1+x)——\frac{1}{2}x^2\)
  5. 积分里面也可以等价无穷小

间断点

  1. 找无定义点【找全最重要】
  2. 左右求极限
  3. 注意三角函数
  4. \(e^x\rightarrow0^+or\rightarrow0^-\)
  5. 绝对值
  • 跳跃间断点、可去间断点，除此以外都是第二类间断点

渐近线

  • 正无穷和负无穷都要考虑

数列极限

两种方法

  • 先求单调性，再求有界——得出极限
  • 先假设有极限，求出极限，证明假设正确

微分

最值与极值

  1. 求导数和二阶导数
  2. 拉格朗日算子（多元函数）
  3. 注意计算仔细则拿分
  4. 注意绝对值，求解时要对定义域进行分类，去绝对值后再求解

可微&可导&连续的关系

  • 可导一定连续，连续不一定可导
  • 连续则极限存在，极限存在不一定连续
  • 连续一定可积，连续一定有界，可积一定有界，可积不一定连续
  • 连续不一定可微，可微一定连续
  • 偏导连续一定可微，偏导存在不一定连续
  • 连续不一定偏导存在，可微不一定偏导连续
  • 二阶混合偏导连续的偏导相等，偏导一个连续一个有界函数可微。

可微*

\[ \lim_{p\rightarrow+\infty}\frac{\Delta f-f_x'(0,0)\Delta x-f_y'(0,0)\Delta y}{\sqrt{\Delta x^2+\Delta y^2}} \]

​ 其中 \[ p={\sqrt{\Delta x^2+\Delta y^2}} \]

  1. 二阶连续偏导数连续 \[ \frac {\partial ^2u} {\partial ^2x }=\frac {\partial ^2u } {\partial ^2y } \]

全微分

\[ du=Pdx+Qdy \]

\[ \rightarrow \frac {\partial P } {\partial y }=\frac {\partial Q } {\partial x } \]

  • 计算可以取对数 \[ u=(\frac{z}{x})^y \]

可导

  1. 不可导点——函数无意义点，分段点
  2. 定义推导，左导数=右导数
  3. 二阶导数——泰勒公式
  4. 题目中若只有连续条件，如果对分段函数求导，分段点一定一定一定要用导数定义，就算连续，但是也不一定可导

中值定理***

  1. 积分再现，三角函数的周期性

    \(sinx=sin(n\pi+x)\)

  2. 求实根问题

    • 单调，\(f(a)\)<0及\(f(b)>0\)
    • 设辅助函数，转换为导数存在为零
      • 导数为零，可用罗尔定理
      • 二阶导数为零，可以使用两次罗尔定理

  3. 中值定理与积分中值定理

  4. 拉格朗日

  5. 泰勒

极值点&拐点

  1. 极值点一阶导函数变号
  2. 拐点二阶导函数变号

积分

计算

  1. 注意积分限，有的积分限会故意颠倒顺序

  2. 积分换元时一定要注意换积分限

  3. 三角函数的积分公式【太多了，每天早中晚默写一遍】 \[ \int tanxdx=-ln|cosx|+c \]

    \[ \int cotxdx=ln|sinx|+c \]

  4. 点火公式推广应用

  5. 当积分上下限确定时，可以当作常数\(A\)看待

  6. 积分限对称问题

    • 关于\(x\)轴对称
    • 关于\(y\)轴对称
    • 关于原点对称
    • 函数的奇偶性
    • 轮换对称性

  7. 积分再现，当出现\(\frac{x^2}{x^2+y^2}\)时可以考虑轮换对称性

  8. 弧长公式 \[ S=\int_b^a \sqrt{1+y'^2}dx \]

  9. 质心公式 \[ \frac{\int_b^a x\alpha (x)dx}{\int_b^a \alpha(x)dx} \]

旋转体问题***

  1. 绕\(x\)轴和\(y\)轴旋转公式的应用
  2. 通用公式\(\int\int_Dr(x,y)dxdy\)
  3. 旋转轴的平移
  4. x、y函数对应关系可以进行转换，将y(x)转换为x(y)

微分方程

  1. 有时候y不一定直接就写成y，可能是一个式子 \[ y'cosy=x-siny \] 此题中可将\(f(x)\)看作\(siny\)

    得,同时令\(siny=u\) \[ （siny）'=x+siny \]

  2. 首先判断微分方程得类型，再决定使用哪种方法计算求解

  3. 微分方程可以是\(\frac{dy}{dx}\)也可以是\(\frac{dy}{dx}\)

  4. 二阶微分方程通解公式

设特解公式

背熟！！

知通解反求微分方程

  • 由通解求\(y',y''\)消去常数\(C_1,C_2\)

幂级数与微分方程

\[ x_n=e^2+\frac{e^4}{3!}+\frac{e^4}{5!}+...+\frac{e^{2n+2}}{(2n+1)!},求lim_{n\rightarrow\infin}x_n \]

\[ 构造\sum_{n=0}^{\infin}\frac{x^n}{n!}=e^x \]

  • 提\(e\)、求导、相加

级数敛散性

  1. P级数
  2. 当如下情况时，需要在不同的情况对函数进行分析，当\(0<x<1\)和\(x>1\)时，占主要作用的部分不同前者为\(b\)，后者为\(a\)

\[ I=\int_{0}^{\infty}\frac{1}{x^a+x^b}dx(a>b>0) \]

不等式的证明

  1. 移到一边
  2. 设辅助函数
  3. 求导
  4. 找特殊值点
  5. 常用不等式的应用

数学应用

  1. 题目中的有关速率的直接写出来
  2. 然后写出需要求解的式子
  3. 判断两者之间的关系，得真正需求求解得式子

线性代数

具体

  1. 求\(A^*\)的方程通解时，通常会转化成与\(A\)相关的方程进行简化求解
  2. 求\(A、B\)正交矩阵，则$P_1AP_1{-1}-\(以及则有\)P_2BP_2{-1}-\(则有\)Q=P_1P_2^{-1}$
  3. 正定矩阵代表特征值不全为0，可以为零。【注意二次型矩阵|A|>0恒成立】
  4. 分块矩阵的计算，必须是拉普拉斯型，角落是\(O\)矩阵

抽象

  1. 构造特征值因式
  2. 矩阵等价\(\Rightarrow\)r相等
  3. 矩阵的逆的求和表现形式
  4. 单位正交列向量\(A^TA=E\)
  5. 反实对称矩阵\(B=-B^T\)
  6. 单位正交向量，\(A^T=A^{-1}\)

线性相关与线性无关

线性相关性与方程组的结合！！！

  • \(A^*A=0\)则\(A\)的列向量是\(A^*\)的解

  1. 不要臆测不要臆测

  2. 但凡线性相关和线性无关都能推导出来

  3. 巧妙使用解方程的思想,用秩

    \(eg.\) \[ \beta_1=\alpha_1+k\alpha_3\\\beta_2=\alpha_2+l\alpha_3\\若\alpha_1,\alpha_2,\alpha_3线性无关，则(\alpha_1,\alpha_2,\alpha_3)是三阶可逆矩阵，故r(\beta_1,\beta_2)=r\\ a = r\left[\matrix{1&0\\0&1\\k&l}\right]=2\\即\alpha_1+k\alpha_3,\alpha_2+l\alpha_3线性无关 \]

  4. 线性无关的延伸组必线性无关

解方程

  1. 通解带k，括号注明

相似对角化条件

  1. 实对称矩阵必相似于对角矩阵
  2. 特征值不同
  3. 有n个不同的特征向量【其他条件算不出来，就计算特征值，看特征向量，肯定能算出来】
  4. 实对称矩阵的特征值必为实数

二次型

  • 配方法，先配一个，再配另一个，按顺序，如果配不出来，说明前面配错了，重算
  • 施密特正交化

  • 两个矩阵经过正定变化之后，正、负惯性指数不变

其他补充

反函数

反函数\(f(x)\)与\(g(x)\)为反函数则\(f(g(x))=x\),但是要注意区分，当\(g(2)\)此处对应的是\(f(x)\)的\(y\),有关\(f(x)\)的\(x\)需要进行求解而得

参数方程

  1. 参数方程二重积分计算时，直接将y带入，再求\(\frac {dx}{dt}\)与替换为\(dx\)

极坐标

  1. 极坐标表示得方程不一定就是角度和半径，也可以是直角坐标系，根绝计算需求选择
  2. $ \(和\)r\(的顺序转换，可以将\) 和r$放入直角坐标系中考虑
